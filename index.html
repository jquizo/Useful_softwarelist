List of Useful Software

All software listed below are either free or have free versions. I recommend starting with Ninite This allows you to install and update your programs all at once.

Web Browsers

  • Chrome - Most popular web browser.
  • Firefox - Another great and popular web browser.
  • Opera - Browser with a built in ad-blocker & VPN.
  • Vivaldi - A browser with many customization options.
  • Brave - Browser that blocks ads and website trackers.
  • TOR - A browser for surfing the web securely and anonymously.

File Viewers

  • 7-Zip - File compression and archiver.
  • ZIP archiver - File compression and archiver.
  • IZArc - File compression and archiver.
  • PeaZip - File compression and encryption.
  • SumatraPDF - Simple PDF viewer.
  • Foxit - PDF viewer.
  • Clover - Adds tabs to Windows Explorer. Compatible with Windows 10.
  • QTTabBar - Adds tabs to Windows Explorer. Compatible with Windows 10.

Audio & Communication

  • Audacity - Audio recording & editor.
  • LMMS - Program for creating your own music tracks.
  • Discord - All in one voice & text chat. Great for games.
  • WhatsApp - Simple, secure messaging.
  • Viber - Free calls and messaging.
  • Pidgin - Chat client.
  • MediaInfo - Shows technical data for video & audio files.
  • MP3TAG - Allows you to edit metadata of audio files.

Media players

  • VLC media player - Multimedia player.
  • Pot Player - Minimalistic media player.
  • Media Player Classic - Lightweight media player.
  • Kodi - Media player and manager.
  • GOM player - Media player.

Video & video editing

  • OBS Studio - Video recording and live streaming.
  • VSDC Video Editor - Simple free video editor.
  • Windows Movie Maker - Free video editor.
  • Hitfilm Express - Free professional grade video editor.
  • Lightworks - Professional grade video editor, free & paid versions.
  • Davinci Resolve 15 - Video, color correction, audio and visual effects editor. Free & paid versions.
  • Aegisub - Subtitle editor.
  • Microsoft Expression Encoder 4 - Screen capture program.
  • Faststone Capture - Screen capture program.
  • HandBrake - Software for converting video formats.

Word Processing & Email

  • LibreOffice - Open source, free alternative to Microsoft Office suite.
  • WPS Office - Free office suite program.
  • AbiWord - Microsoft word alternative. Word processing program.
  • Bullzip - PDF printer.
  • PDFCreator - Converts documents to PDF.
  • Thunderbird - Email application.
  • Mailbird - Email client. Free and paid versions.
  • pCloud - For transferring large files through email. Upto 5GB.
  • Dropsend - For transferring large files through email.

Tools

  • f.lux - Reduces blue glow in computer screens. Makes the color of your screen adapt to the time of day.
  • TeraCopy - File transfer utility. Copies files faster than Windows.
  • VoidTools Everything - Windows search replacement. Finds everything.
  • Greenshot - Print screen tool.
  • qBittorrent - Torrent client.
  • uTorrent - Torrent client.
  • FreeNAS - Storage Operating System. Creates a centralized place to access your data.
  • Folder Size - Shows folder size.
  • CCleaner - File and registry cleaner.
  • BleachBit - Alternative to CCleaner. Frees up disk space.
  • dBan - Hard Drive eraser and data clearing program.
  • WinDirStat - Disk usage viewer.
  • Rainmeter - Desktop customization program.
  • AutoHotkey - Program for automating tasks in Windows.

Security

  • Avira - Lightweight Antivirus. Free and paid options.
  • Kaspersky - Antivirus. Free and paid options.
  • AdwCleaner - Adware cleaner & Removal Tool.
  • MalwareBytes - Antimalware software, finds and removes malware. Free and paid options.
  • pfSense - Open source and free firewall.
  • Authy - 2 Factor authenticator.
  • uBlock Origin - Lightweight adblocker for Chrome.
  • AdBlock - Ad blocker alternative, although uses more system resources than uBlock Origin.
  • Privacy badger - Browser extension for Chrome, Firefox, Opera. Automatically blocks invisible trackers.
  • Panda USB vaccine - Security utility that blocks malware from spreading through USB drives.
  • VeraCrypt - Encryption software.
  • AxCrypt - Encryption software. Free and paid versions.
  • HTTPS Everywhere - Encrypts your communication with websites. Forces https when it can.
  • Windscribe - Free VPN, Pro version available.

Hardware Information & Monitoring

  • Speccy - Shows your computer specs.
  • HWMonitor - Hardware monitoring program that tracks temperatures, fans, voltages and speed.
  • Open Hardware Monitor - Hardware monitoring program that tracks temperatures, fans, voltages and speed.
  • HWiNFO - In-depth hardware monitor and reporter.
  • CPU-Z - Provides CPU, Motherboard and RAM information.
  • GPU-Z - Provides GPU information.
  • HDDscan - Hard drive information and diagnostics. Useful for checking the health of your hard drives.
  • CrystalDiskInfo - HDD/SSD utility software.
  • MSI Afterburner - Provides detailed overview of your hardware and has an overclocking utility.
  • AIDA64 - For CPU & RAM stability and benchmarking.
  • LinX - CPU benchmarking app.
  • WinMTR - Network Diagnostic tool.
